Connect to GitHub

All registration data saves as a JSON file directly in your GitHub repo. Works on any device, any browser. Anyone you share the token with can edit.

  1. Go to github.com/settings/tokens/new
  2. Name it "RWO Registration" — set expiration to No expiration
  3. Check the top-level repo checkbox
  4. Click Generate token and paste it below
